## Deploy to Vercel

This project is ready to deploy to Vercel!

### Quick Deploy

1. Install Vercel CLI (if you haven't already):
   ```bash
   npm install -g vercel
   ```

2. Deploy:
   ```bash
   vercel
   ```

3. Follow the prompts and your site will be live!

## Local Usage

Simply open `index.html` in your browser.
